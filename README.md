# Executando testes no projeto Enjoeat



### Instalação
Ruby 2.5

Install Chrome 59
chromedriver


**Instalar o Bundle**
```
gem install bundler
```


**Clonar o projeto**
``` 
git clone https://github.com/julio958214/Enjoeat.git
```

**Instalar dependências**

```
bundle install
```
